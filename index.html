Funções de Arrays em JavaScript

1. splice()

A função splice() altera o conteúdo de um array removendo ou substituindo elementos existentes e/ou adicionando novos elementos. 


const frutas = ['maçã', 'banana', 'laranja'];
frutas.splice(1, 1, 'manga'); // Remove 1 elemento na posição 1 e adiciona 'manga'
console.log(frutas); // ['maçã', 'manga', 'laranja']

2. concat()

A função concat() é usada para unir dois ou mais arrays. Ela retorna um novo array sem alterar os arrays originais. 


const array1 = [1, 2, 3];
const array2 = [4, 5, 6];
const arrayConcatenado = array1.concat(array2);
console.log(arrayConcatenado); // [1, 2, 3, 4, 5, 6]

3. includes()

A função includes() verifica se um array contém um determinado elemento, retornando true ou false. 


const numeros = [1, 2, 3, 4, 5];
const temTres = numeros.includes(3); // true
console.log(temTres); // true

4. filter()

A função filter() cria um novo array com todos os elementos que passam em um teste definido por uma função fornecida. 


const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(num => num % 2 === 0);
console.log(pares); // [2, 4]

5. find()

A função find() retorna o primeiro elemento que satisfaz a condição fornecida em uma função. Se nenhum elemento for encontrado, retorna undefined. 


const numeros = [1, 2, 3, 4, 5];
const primeiroPar = numeros.find(num => num % 2 === 0);
console.log(primeiroPar); // 2
